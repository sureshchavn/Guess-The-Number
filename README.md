# Guess-The-Number
Interactive browser-based number guessing game built using HTML, CSS, and JavaScript. Features multiple difficulty levels, dynamic score calculation, attempt tracking, guess history, input validation, and restart functionality. Designed with responsive UI and clean game logic for an engaging user experience.

 Guess The Number Game

An interactive browser-based number guessing game built using HTML, CSS, and Vanilla JavaScript.
Players select a difficulty level and attempt to guess the randomly generated secret number within limited attempts.
